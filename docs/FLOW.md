1. The Researcher ("The Librarian")
   Job: Go out to the internet, find the best books (URLs), buy them, and put them on the shelf (Ingest to Database).
   Output: "I have stocked the library with 7 books on this topic. Here is a list of what acts are covered." (The Brief).
   It does NOT write the article.
2. The Generator ("The Writer")
   Job: Read the Brief. Go to the library (Database), look up specific pages (Retrieve), and write the manuscript.
   Output: A Draft Article (file.md).
   It does NOT go to the internet.
3. The Auditor ("The Editor")
   Job: Read the Draft. Check against the Style Guide (Rubric). Critically roast it.
   Output: A Score & Fix-it List.
4. The Reviser ("The Polisher")
   Job: Read the Editor's notes. Fix the Draft.
   Output: Final Article (file.md).
